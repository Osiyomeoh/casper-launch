# Security Policy

## Supported Versions

| Version | Supported |
|---|---|
| testnet (current) | Yes |

## Reporting a Vulnerability

Please do **not** open a public GitHub issue for security vulnerabilities.

Email: samuelaleonomoh5@gmail.com

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (optional)

You will receive a response within 48 hours. If confirmed, a fix will be prioritized and you will be credited in the release notes.
